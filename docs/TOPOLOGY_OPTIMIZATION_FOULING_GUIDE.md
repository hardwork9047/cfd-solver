# 膜ファウリング抑制に向けたトポロジー最適化の適用指針

この文書は、LBM-DEM による膜細孔内粒子輸送シミュレーションを、円柱形状・配置の最適化へ発展させるための指針をまとめたものです。

重要な前提として、粒子間付着、凝集、架橋、ケーキ層形成、再剥離は非線形かつ履歴依存性が強く、単純なサロゲートモデルで完全に置き換えるのは難しいです。そのため、サロゲートモデルは「実ファウリング挙動の完全な代替」ではなく、設計候補を絞り込むための高速なリスク評価器として使います。

## 基本方針

最初からトポロジー最適化を直接適用するのではなく、以下の順番で進めます。

1. 多数の LBM-DEM 計算を行う
2. 各計算から流れ場、粒子挙動、接触、流束、圧力損失などの指標を抽出する
3. ファウリング指標との相関関係を調べる
4. サロゲートモデルに使う入力特徴量と目的変数を決める
5. サロゲートで候補形状を高速に評価する
6. 上位候補だけ LBM-DEM で再検証する
7. 新しい計算結果をデータセットに追加し、サロゲートを更新する

この流れは active learning 型の設計探索として扱います。

```text
候補形状を生成
→ サロゲートで高速評価
→ 良さそうな候補・不確実な候補を選ぶ
→ LBM-DEM 本計算で検証
→ データセットに追加
→ サロゲートを再学習
```

## サロゲートモデルの役割

サロゲートモデルが予測すべきものは、最終的な付着粒子数そのものだけではありません。むしろ初期段階では、以下のような「ファウリングしやすさ」を表す粗視化指標を予測する方が安全です。

- 圧力損失
- 透過流束
- 低速域の大きさ
- 低せん断域の大きさ
- 粒子の滞留しやすさ
- 円柱・壁との接触頻度
- 粒子間接触頻度
- 閉塞リスク
- 流路の偏り

粒子間付着の詳細、凝集、架橋、ケーキ層形成は、最終的には LBM-DEM 本計算で検証します。

## 目的変数の候補

膜ファウリングを単一指標で表すのは危険です。最初は複数の目的変数を並べて解析します。

候補となる目的変数:

- `final_total_contacts`: 最終時刻の総接触点数
- `mean_total_contacts`: 時間平均の総接触点数
- `contact_growth_rate`: 接触点数の増加速度
- `final_particle_area_fraction`: 最終粒子面積分率
- `particle_area_fraction_growth_rate`: 粒子堆積の増加速度
- `final_passed_particles`: 通過粒子数
- `passed_particle_ratio`: 投入粒子に対する通過率
- `mean_pressure_drop`: 平均圧力損失
- `pressure_drop_slope`: 圧力損失の増加速度
- `mean_permeate_flux`: 平均透過流束
- `flux_decline_rate`: 透過流束低下率
- `clogging_time`: 閉塞または急激な流束低下が起こる時刻

総合ファウリングスコアを作る場合は、各量を正規化してから重み付き和にします。

```text
fouling_score =
  + a * normalized(contact_growth_rate)
  + b * normalized(pressure_drop_slope)
  + c * normalized(final_particle_area_fraction)
  - d * normalized(mean_permeate_flux)
  - e * normalized(passed_particle_ratio)
```

重み `a, b, c, d, e` は、研究目的に応じて変えます。たとえば「閉塞抑制」を重視する場合は接触増加と圧力損失を大きくし、「処理量維持」を重視する場合は透過流束を大きくします。

## 説明変数の候補

サロゲートの入力には、幾何特徴量、流れ場特徴量、粒子挙動特徴量を分けて入れます。

### 幾何特徴量

- 円柱数
- 各円柱の半径
- 固体面積率
- 最小細孔幅
- 平均細孔幅
- 円柱間距離の最小値・平均値
- 入口から出口までの開口率
- 流路の曲がり度
- 配置の千鳥度
- 円柱配置の左右・上下非対称性
- 円柱表面積の合計
- 流路ボトルネック数

幾何特徴量は、流れ場や粒子計算をしなくても計算できます。候補形状を大量にふるい落とす初期フィルタとして有用です。

### 流れ場特徴量

LBM のみ、または LBM-DEM の初期段階から抽出します。

- `mean_ux`
- `max_speed`
- `mean_pressure`
- `pressure_drop`
- `permeate_flux`
- 低速域面積
- 低せん断域面積
- 再循環領域面積
- 円柱近傍の平均速度
- 円柱近傍のせん断速度
- 壁面せん断応力分布
- 圧力勾配の局所集中

ファウリングは低速域、低せん断域、再循環域で進みやすいため、これらは重要な候補特徴量です。

### 粒子挙動特徴量

LBM-DEM の時系列から抽出します。

- `passed_particles`
- `particle_particle_contacts`
- `wall_contacts`
- `cylinder_contacts`
- `total_contacts`
- `particle_area_fraction`
- `particle_ke`
- `mean_force`
- 粒子の平均滞留時間
- 円柱近傍の滞在時間
- 粒子-円柱接触頻度
- 粒子-粒子接触頻度
- 再剥離回数
- クラスター形成頻度

粒子間付着挙動は直接サロゲート化が難しいため、接触頻度や滞留時間のような粗視化指標として扱います。

## 相関解析

サロゲートを作る前に、まずは指標間の相関関係を洗い出します。

使う解析:

- Pearson 相関
- Spearman 相関
- 相互情報量
- ランダムフォレストの特徴量重要度
- SHAP 値
- PCA / UMAP による条件クラスタリング
- 時系列のラグ相関

特に重要なのは、同時刻の相関だけでなく時間遅れを見ることです。

```text
早い時刻の低速域・接触頻度・滞留時間
↓
後の圧力損失上昇・流束低下・接触点増加
```

このようなラグ相関を見ることで、ファウリング進行の前兆指標を探します。

## サロゲートモデルの候補

初期段階では、データ数が少ないためシンプルなモデルから始めます。

推奨順:

1. Random Forest
2. Gradient Boosting / LightGBM / XGBoost
3. Gaussian Process Regression
4. Bayesian Neural Network
5. CNN / U-Net / Fourier Neural Operator

最初から深層学習モデルを使う必要はありません。まずは表形式の特徴量とツリーベースモデルで、どの指標が効いているかを見る方が有益です。

データが増えて、流れ場や固体マスクそのものを入力にしたくなった段階で、CNN、U-Net、FNO などを検討します。

## トポロジー最適化への接続

サロゲートモデルができたら、次の2段階でトポロジー最適化に接続します。

### 段階1: パラメータ最適化

円柱を明示的な設計変数として扱います。

```text
design = [
  x1, y1, r1,
  x2, y2, r2,
  ...
  xN, yN, rN
]
```

制約:

- 円柱同士が重ならない
- 壁と重ならない
- 最小細孔幅を確保する
- 固体面積率を上限以下にする
- 入口・出口を完全に塞がない

探索アルゴリズム:

- Latin Hypercube Sampling
- Bayesian Optimization
- CMA-ES
- Genetic Algorithm
- NSGA-II

この段階では、サロゲートで多数候補を評価し、上位候補だけ LBM-DEM で検証します。

### 段階2: 連続トポロジー最適化

設計領域に連続変数 `gamma(x, y)` を置きます。

```text
gamma = 0: 流体
gamma = 1: 固体
```

LBM 側には Brinkman 型の抵抗項を入れます。

```text
F_solid = -alpha(gamma) * u
```

SIMP 型の補間を使うと、中間密度を抑制できます。

```text
alpha(gamma) = alpha_max * gamma^p
```

この連続設計で得た固体分布を、最終的に円柱・楕円・支柱・リブなどの製造可能な形状へ変換します。

## 推奨ワークフロー

このリポジトリで進める場合の実装順は以下です。

1. `run_lbm_dem.py` で多数の円柱配置を計算する
2. 各条件の `analysis/time_series.csv` を集める
3. 円柱配置情報を条件ごとに保存する
4. 幾何特徴量を計算する
5. 時系列から集約特徴量を作る
6. ファウリング目的変数を複数定義する
7. 相関解析を行う
8. サロゲートモデルを学習する
9. 候補円柱配置を大量生成する
10. サロゲートで候補をランキングする
11. 上位候補を LBM-DEM で再計算する
12. 結果をデータセットに追加する

## 注意点

サロゲートモデルで得た最適形状を、そのまま最終解として扱ってはいけません。

サロゲートは、あくまで高価な LBM-DEM 計算の回数を減らすための案内役です。最終的な候補は必ず、粒子間引力、斥力、転がり摩擦、粒子体積分率を含む LBM-DEM 本計算で検証します。

また、相関が強い指標が原因であるとは限りません。幾何、流れ場、粒子挙動の特徴量を同時に入れ、支配因子を慎重に分けて解釈する必要があります。

## 最初に目指すべき成果

最初のゴールは、完全なトポロジー最適化ではなく、以下を明らかにすることです。

- ファウリングと強く相関する指標は何か
- 圧力損失とファウリング抑制のトレードオフはどこにあるか
- 円柱数、半径、配置のどれが支配的か
- 低速域、低せん断域、接触頻度のどれが前兆指標として有効か
- サロゲートで候補をどの程度ふるい落とせるか

この基盤ができてから、Brinkman/SIMP 型の連続トポロジー最適化へ進むのが安全です。
